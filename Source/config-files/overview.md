# Code On Tap Configuration Files

## Configuration Files

### domains.json

* Repository - AccountCMDB
* Scope - Tenant
* Type - Config
* Example:
    ````JSON
    {
        "Domains" : {
            "Validation" : "codeontap.io",
            "cot" : {
                "Stem": "codeontap.io"
            },
            "syd1" : {
                "Stem": "syd1.codeontap.io"
                }
            }
    }
    ````

### tenant.json

* Repository - AccountCMDB
* Scope - Tenant
* Type - Config
* Example:
    ````JSON
    {
        "Tenant" : {
            "Title" : "Code on Tap",
            "Id" : "cot",
            "Domain" : "cot"
        },
        "Account" : {
            "Region" : "ap-southeast-2",
            "Domain" : "syd1"
        },
        "Product" : {
            "Region" : "ap-southeast-2",

                "Region" : "us-west-2"
            }
        },
        "Segment" : {
            "SSH" : {
                "IPAddressGroups" : ["cbroffice"]
            }
        }
    }
    ````
